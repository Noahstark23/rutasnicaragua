# rutasnicaragua

Aplicaci\u00f3n de ejemplo con autenticaci\u00f3n basada en Node.js. Incluye registro y login de usuarios con validaciones b\u00e1sicas y generaci\u00f3n de token JWT.

## Inicio r\u00e1pido

1. Instala las dependencias con `npm install`.
2. Copia `.env` y actualiza las variables de conexi\u00f3n a MongoDB y la clave secreta.
3. Ejecuta `npm start` para iniciar el servidor.

La API expone `/api/auth/register` y `/api/auth/login`.
